# ready

----

[![spm version](http://spmjs.io/badge/ready)](http://spmjs.io/package/ready)


Just Go you scripts after ready.

Support browsers:

* IE6+
* Firefox 2+
* Safari 3+
* Chrome *
* Opera *

## Install

```
$ spm install ready --save
```

## Usage

```js
var ready = require("ready");
ready(function(){
  console.log("DOM Ready!");
});
```
